  1. ¿Que es un sistema de control de versiones?

    Es un sistema que registra los cambios que se realizan en un archivo o en un conjunto de archivos a lo largo del tiempo para que puedas recuperar versiones específicas más adelante

  2. ¿Cual es la diferencia entre git y github?

    Git es la herramienta para realizar el seguimiento de los cambios, mientras que GitHub es el lugar para almacenar los proyectos y trabajar en equipo

  4. ¿Cómo clono un repositorio?

    1º Nos metemos en el repositorio que queramos copiar

    2º Luego hacemos un fork en el repositorio que hayamos copiado

    3º Le damos a code y copiamos el SSH y pones en la terminal el comando "git clone (el SSH)"

  5. ¿Cómo puedo bajar los ultimos cambios de un repositorio?

    Debes de escribir los siguientes comando en el siguiente orden

    1º git status

    2º git add .

    3º git commit -m "commentario"

    4º git push

  6. ¿Cómo puedo gener un tag?

    Haz clic en tu cuenta, arriba a la izquierda, y ve a Configuración.

    En el apartado "Más", haz clic en Gestionar tags.

    Haz clic en "+ Añadir tag".

    Asigna un nombre al nuevo tag, y pulsa el botón "Crear".

  7. Cómo hago una mezcla de ramas?

    Use git checkout para cambiar a la rama en la que desea fusionar. Esta rama suele ser la rama principal. A continuación, use git merge y especifique el nombre de la otra rama que desea incorporar a esta rama

  8. ¿Cómo puedo moverme a un commit especifico?

    Puedes volver a un commit anterior mediante git reset --soft (o --hard, aunque es peligroso :-P) <"commit-adonde-regresas">, o mediante git checkout <"commit-adonde-regresas">

  9. ¿Cómo puedo borrar una rama?

    En el menú Git de la barra de menús, elija Administrar ramas. En Ramas, haga clic con el botón derecho en una rama y seleccione Eliminar

  10. ¿Cómo puedo borrar un tag?

    Al usar la opción -d y un identificador de etiqueta de git tag

  11. ¿Cómo puedo ver los cambios que estan por enviarse al repositorio?

    Con el comando git status

  12. ¿Para que sirve el comando git remote -v?

    Administra conexiones remotas

  13. ¿Cómo le digo a git cual es mi email y correo?

    Tienes que poner el comando: git config --global -user.email "[correo]"

  14. ¿Cuales son los pasos para conectar git con github?

    Debes de generar una clave SSH publica

  15. ¿Cómo puedo crear un alias en git?

    Los alias de Git se crean mediante el comando git config
